# E-commerce_multimodal_recommendation
一个电商超长序列多模态推荐系统
基于Taobao-MM数据集（139GB）和你的硬件配置（16GB内存、6GB显存），我重新设计了这份**以「采样迭代+流式读取」为核心策略**的一周实施计划。

------

## 一、项目概述

**项目名称**：基于Taobao-MM数据集的超长序列多模态推荐系统

**核心目标**：复现MUSE（MUltimodal SEarch-based framework）框架的核心思想，在Taobao-MM数据集上完成用户**超长行为序列**（最长1000条）的建模与CTR预估。

**数据集简介**（来自官网taobao-mm.github.io）：

| 统计项          | 数值                                  |
| --------------- | ------------------------------------- |
| 用户数          | 879万                                 |
| 商品数          | 3540万                                |
| 样本总数        | 9900万（训练集7600万 + 测试集2300万） |
| 行为序列长度    | 最长1000条                            |
| 多模态Embedding | 128维，int8量化                       |
| 存储大小        | 约139GB                               |

**文件结构**：

- `train_samples.parquet` / `test_samples.parquet`：样本（user_id, item_id, label）
- `train_user_features.parquet` / `test_user_features.parquet`：用户特征（含行为序列）
- `item_features.parquet`：商品特征
- `scl_embeddings_int8_p90.parquet`：商品多模态Embedding查找表

**MUSE核心洞察**：

- **GSU（通用搜索单元）** ：轻量级余弦相似度检索Top-K历史行为，追求简单高效
- **ESU（精确搜索单元）** ：丰富的多模态序列建模 + ID-多模态融合，追求精准

**关键代码**（懒加载Dataset）：

```python
class TaobaoMMDataset(Dataset):
    def __init__(self, sample_path, user_seq_dict, item_emb_dict, nrows=None):
        self.df = pd.read_parquet(sample_path, nrows=nrows)  # 只加载采样
        self.user_seq_dict = user_seq_dict
        self.item_emb_dict = item_emb_dict
    
    def __getitem__(self, idx):
        row = self.df.iloc[idx]
        user_id, item_id, label = row['129_1'], row['205'], row['label_0'][0]
        seq = self.user_seq_dict.get(user_id, [0]*1000)[:1000]  # 截断至1000
        emb = self.item_emb_dict.get(item_id, np.zeros(128))
        return seq, emb, label
```
